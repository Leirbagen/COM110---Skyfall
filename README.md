# COM110---Skyfall
A 2D fast-paced arcade game focused on infinite level and dynamic jumping


A physics-driven web game built from scratch using Vanilla JavaScript and HTML5 Canvas.

Welcome to Skyfall! This is a personal game development project created to explore interactive design, custom physics calculations, and robust software architecture without the use of external game engines. 
<img width="1200" height="677" alt="image" src="https://github.com/user-attachments/assets/d0f93914-e926-44ec-8326-f79084e9743c" />
The primary goal of this project was to apply advanced **Object-Oriented Programming (OOP)** concepts to game development. By building the game loop and rendering engine from the ground up, this project demonstrates a deep understanding of:
* **Encapsulation & Inheritance:** Structuring game entities cleanly and efficiently.
* **Custom Physics Engine:** Handling gravity, precise jumping mechanics, and collision detection mathematically.
* **HTML5 Canvas Rendering:** Managing frame-by-frame drawing and real-time screen updates.
